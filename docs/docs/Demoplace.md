# Demo place

There is a demo place that showcases how to create basic logic and functionality using jTDF

You can find it <a href="https://www.roblox.com/games/125363349404346/Bajs-Tower-Defense-Framework-Demo">on this roblox page</a>. The game is uncopylocked.

Features implemented in the demo place:

- Enemy rendering
- Tower upgrades
- Example enemy path
- Data replication
- Example implementation of several tower types (solo, burst, multi enemy, freeze, etc.)
